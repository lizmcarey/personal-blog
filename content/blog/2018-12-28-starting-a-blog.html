I finally started a blog!

I first thought about starting a blog 3 years ago. I never went through it because:

  1. I didn’t know where to begin
  2. I couldn’t think of any content
  3. Laziness

The trick is to set aside a few hours and commit.

If you’re familiar with R, the only resource you need is Yihui Xie’s blogdown book.

Here are the steps I took to publish this blog:

  • Found a blog theme I liked
    Browsing through Hugo’s template collections was overwheleming. Instead, I went through the personal websites of rstats profiles I currently follow (e.g. Yihue Xie, Hadley Wickham, Andrew Heiss, David Robinson, Mara Averick). I settled on the Cocoa Enhanced theme on Julia Silge’s website.

  • Init git and deployed through Netlify
    Take the time to create a repo on Github and link your local files. Being able to push changes through git and have it continously deployed through Netlify is a huge timesaver.
    It took me awhile to get Netlify to automatically deploy changes from my repo. I found out it’s because the default build command on Hugo is too old, you’ll need to create an environment variable HUGO_VERSION with a Hugo version greater or equal to 0.20.
    You can read more here.

  • Used the Serve Site add-in
    The Serve Site add-in lets you preview edits you’ve made in the Viewer pane in Rstudio. I found this super helpful in helping gudie any major changes to the website.

You can find a well documented post on how to publish your own website on Alison Hill’s website.
